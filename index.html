FAQ

How do I run BENG-UI in my Roblox?

We recommend using an editor to upload your SCRIPT source code to the library for execution. Some mobile executors cannot be executed from source code; only desktop executors support this. However, Solara, Xeno, and JJsplot are not supported due to missing functions.

Does BENG - UI support mobile and client versions?

Yes, BENG - UI supports both mobile and client-side applications. We've added a detection mechanism to the script.

Does it support custom themes?

Yes, it supports built-in themes and customization. You can also customize using code; simply enter the UI color code you want. Hexadecimal is currently supported.
